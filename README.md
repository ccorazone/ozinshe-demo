# ÖZINŞE Demo 

Мобильное приложение для онлайн-кинотеатра на казахском языке. Проект разработан в рамках изучения мобильной разработки на iOS.

## Особенности проекта
* **Верстка кодом:** Использование библиотеки **SnapKit** для создания интерфейса.
* **Темная тема:** Поддержка Dark Mode с сохранением выбора пользователя через **UserDefaults**.
* **Сетевой слой:** Реализация запросов к API через **Alamofire** и обработка JSON-данных.
* **Onboarding:** Реализация экранов приветствия с использованием **UIPageViewController**.

## Технологический стек
* **Language:** Swift 5
* **Layout:** SnapKit (Auto Layout)
* **Networking:** Alamofire, URLSession
* **UI Components:** UITableView, UIPageViewController, SVProgressHUD
* **Data Persistence:** UserDefaults

## Структура проекта
Код организован по логическим папкам для удобства поддержки:
* `Model/` — Модели данных и сетевые сущности.
* `View/` — Кастомные элементы интерфейса и ячейки.
* `ViewController/` — Контроллеры экранов.
* `ViewModel/` — Логика подготовки данных.
